討論、留言介面

提供會員間互動管道。基本要求

  • 可以留言。
  • 留言可以用 markdown 排版。
  • 摘要留言者的身份。
  • 回覆特定留言。成串留言。
  • 刪除 / 編輯留言

  • 管理者:

    • 刪除留言 / 隱藏留言 / 編輯留言
    • 鎖定討論串

其它可能要做的

  • 新留言通知
  • 討論的 dashboard
  • 全活動留言管理
  • 留言狀況視覺化？
