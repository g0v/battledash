活動後台頁面

入口:
  • 個人檔案
  • 活動資訊頁
  • 活動提案列表
出口
  • 活動提案列表
  • 活動資訊頁
  • 活動資訊頁編輯器
主要功能:
  • 預覽活動
  • 修改活動資訊

  • 調整活動狀態

    • 開關報名 / 尚未開跑 / 活動中 / 活動結束
    • 開關編輯 / 開關評論 / 開關評審
    • 設定階段 ( 多階段活動用 )
    • 發布活動 ( 發布時檢查所有資訊是否完整 )

  • 提案資訊

    • 提案數
    • 評論狀況
  • 角色調整
  • 刪除活動/隱藏活動

活動 schema

activity description - text(300) title - text(100) slug - text(32) not null unique key - int not null unique primary key deleted - bool visible - bool detail: jsonb createdtime - timestamp modifiedtime - timestamp orgnization - int foreign key(organization) owner - int foreign key(user) - index key, slug, createdtime
